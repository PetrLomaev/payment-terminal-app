# Project "Mobile phone payment terminal"

## Project Description

The following usage functions are available:

- Includes a main screen with a list of operators and a payment form screen

- The payment form screen contains the selected operator, a field for entering a phone number with a mask and validation, and a field for entering the amount in rubles

- After sending the payment, a success or error message is displayed (API calls are emulated, the response is random). If successful, proceed to the main screen


## Technologies used

- React, React Hooks

- TypeScript

- Next.js

- Styled-Components


## Minimum System Requirements

Node.js Command Line, npm


## Install 

`git clone git@github.com:PetrLomaev/payment-terminal-app.git`

`cd payment-terminal-app`

`npm install`

`npm run dev`

Open your browser and go to: http://localhost:3000
